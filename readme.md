# Drawing App

**Drawing App** is a lightweight and interactive sketching playground built using **HTML**, **CSS**, and **Vanilla JavaScript**. It allows users to draw freely with customizable colors, erase parts of their sketch, or reset the entire canvas—all within a smooth and responsive interface.

**Website**: [Drawing App](https://drawing-app-chi-brown.vercel.app/)

---

## Features

- **Custom Color Selection** – Pick any color to draw.
- **Eraser Tool** – Erase parts of your sketch easily.
- **Reset Canvas** – Clear the entire drawing in one click.
- **Smooth & Responsive** – Fast, fluid drawing experience across devices.
- **Minimalist UI** – Simple and intuitive design for creative freedom.

---

## Tech Stack

- **HTML** – Markup
- **CSS** – Styling
- **JavaScript** – Drawing logic & interactivity
